# デプロイ・更新ガイド (GitHub連携版)

このプロジェクトは **GitHub** と **Vercel** が連携しており、ローカルで修正してプッシュするだけで、自動的に本番環境が更新されます。

## 修正から反映までの手順

日々の開発サイクルは以下の通りです。

1.  **コードを修正する**
2.  **変更をコミットしてプッシュする**
    ターミナルで以下の3つのコマンドを実行します。

    ```powershell
    git add .
    git commit -m "修正内容のメモ（例：求人詳細ページのレイアウト修正）"
    git push
    ```

3.  **自動デプロイの完了を待つ**
    *   `git push` を実行すると、Vercelが自動的に検知してビルドを開始します。
    *   通常 1〜2分 で完了し、公開URL（Vercelのダッシュボードで確認可能）に最新の内容が反映されます。

## 公開URLの確認方法

*   [Vercel Dashboard](https://vercel.com/dashboard) にアクセスし、プロジェクト（`digital-guild-mvp`）をクリックすると、現在の公開URLが表示されています。
*   `Visit` ボタンを押すと、実際のサイトが開きます。

---
**補足**:
もう `npx vercel` コマンドを使う必要はありません。すべて `git push` だけで完結します。
