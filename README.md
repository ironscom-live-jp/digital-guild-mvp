# Digital Guild System (Demo Mock)

Digital Guild SystemのMVP（Minimum Viable Product）デモモックへようこそ。
このプロジェクトは、AIとブロックチェーンを活用した新しい短期労働マッチングプラットフォームのコンセプトモデルです。

## 🚀 セットアップ (Setup)

このデモをローカル環境で実行するには、Node.jsが必要です。

1.  **リポジトリのディレクトリへ移動**
    ```bash
    cd digital-guild-system
    ```

2.  **依存関係のインストール**
    ```bash
    npm install
    ```

## ▶️ 起動方法 (How to Start)

1.  **開発サーバーの起動**
    ```bash
    npm run dev
    ```

2.  **ブラウザでアクセス**
    ブラウザを開き、以下のURLにアクセスしてください。
    [http://localhost:3000](http://localhost:3000)

## 🌍 デプロイ (Deployment)

このプロジェクトは **Firebase App Hosting** へのデプロイに対応しています。
Next.js アプリケーションは、GitHub リポジトリ経由で自動的にビルド・デプロイされます。

### 手順概要
1. GitHub にリポジトリを作成し、コードをプッシュします。
2. Firebase Console で App Hosting を設定し、GitHub リポジトリと連携します。

詳細な手順については、[デプロイガイド (firebase_guide.md)](./firebase_guide.md) を参照してください。

## 📖 デモシナリオ (Demo Walkthrough)

このデモでは、**Worker（働き手）** と **Provider（事業者）** の2つの役割を体験できます。
認証はモック化されており、ウォレット接続ボタンを押すだけでログイン状態になります。

### 共通操作: ウォレット接続
1.  画面右上の「ウォレット接続」ボタンをクリックします。
2.  数秒で接続が完了し、デモ用の残高（SC: Stable Coin）が表示されます。

---

### 👨‍🌾 Worker（働き手）としての体験

働き手として、仕事を探し、応募し、報酬を受け取る流れを体験します。

1.  **仕事を探す**
    - メニューの「仕事を探す」をクリック、またはトップページのCTAボタンから移動します。
    - 募集中の仕事一覧が表示されます。

2.  **仕事に応募する**
    - 気になる仕事（例：「高原野菜の収穫」）をクリックして詳細画面を開きます。
    - 「応募する」ボタンをクリックします。
    - 確認モーダルで「応募を確定する」をクリックすると、応募が完了します。

3.  **ステータスの確認**
    - メニューの「マイページ」に移動します。
    - 「現在のジョブ」セクションで、応募した仕事のステータスが確認できます。
    - 事業者が承認すると、ステータスが「承認済み」に変わります（※デモでは事業者画面で操作が必要）。

4.  **業務完了報告**
    - 承認された仕事に対して「完了報告」ボタンが表示されます。
    - ボタンをクリックし、報告を行うと、報酬がウォレットに反映されます。

---

### 🏢 Provider（事業者）としての体験

事業者として、求人を出し、応募者を承認し、報酬を支払う流れを体験します。

1.  **事業者ダッシュボードへアクセス**
    - 以下のURLに直接アクセスするか、メニューにはない隠しリンク（デモ用）として `/dashboard/provider` にアクセスします。
    - [http://localhost:3000/dashboard/provider](http://localhost:3000/dashboard/provider)

2.  **求人を掲載する**
    - ダッシュボード左側の「新規求人作成」フォームに入力します。
    - タイトル、報酬額、募集人数などを入力し、「求人を公開」をクリックします。
    - 右側の「掲載中の求人」リストに新しい求人が追加されます。

3.  **応募者の承認**
    - Workerが仕事に応募すると、ダッシュボード右側の「承認待ちリスト」に応募が表示されます。
    - 「承認」ボタンをクリックすると、Workerの応募が承認され、スマートコントラクト（モック）がデポジットをロックします。

## ⚠️ 注意事項 (Notes)

- **データの一時性**: このデモはデータベースに接続されていません。ページをリロードすると、投稿した求人や応募状況、ウォレット残高はリセットされます。
- **ウォレット**: 実際のブロックチェーンウォレット（Metamask等）は使用しません。内部的なステート管理のみで動作します。
